# Entia Futura Institut

Official website project for 未来人研究所 / ENTIA FUTURA INSTITUT.

## Project

- Site: `https://entiafutura.com`
- Stack: `Astro + Cloudflare Workers + cloudflared`
- Project directory: `/Users/zhourongchang/project_cs/projects/entia-futura`

## Commands

```sh
npm install
npm run dev
npm run build
npm run cf:dev
npm run cf:deploy
npm run tunnel
```

## Development

1. `cd /Users/zhourongchang/project_cs/projects/entia-futura`
2. `npm run dev`
3. Optionally run `npm run tunnel` for a public preview URL

## Notes

- `wrangler.jsonc` is configured for Cloudflare Workers.
- Brand assets live in `public/brand/`.
